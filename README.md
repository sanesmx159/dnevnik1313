# Школьный-дневник (проект для Render)

Проект содержит простой Node.js сервер и фронтенд для электронного дневника.

## Как запустить локально
1. Установи зависимости: `npm install`
2. Запусти: `npm start`
3. Открой в браузере: `http://localhost:3000`

## Как задеплоить на Render
1. Создай репозиторий на GitHub и запушь все файлы (включая папку public, server.js, data.json, package.json).
2. Перейди на https://render.com и создай новый **Web Service**.
3. Выбери GitHub-репозиторий.
4. В поле **Build Command** оставь пустым или `npm install`.
5. В поле **Start Command** укажи `npm start`.
6. Нажми Deploy — через пару минут твой сайт будет онлайн.

Пароль учителя: `1324576800` (можно изменить в server.js)
